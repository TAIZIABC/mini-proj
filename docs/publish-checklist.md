## 📈 发布后 48 小时关注指标

| 指标 | 达标线 | 备注 |
|---|---|---|
| 完读率 | > 30% | 公众号看「阅读完成比例」 |
| 收藏/阅读 | > 5% | 高质量技术文的核心指标 |
| 转发/阅读 | > 2% | 决定二次传播 |
| 评论质量 | 出现具体问题 | 说明读者真的在尝试落地 |

**低于达标线的话**：先看完读率。
- 完读率低 → 标题 OK，内容节奏有问题（前 3 段不够吸引）
- 完读率 OK 但收藏低 → 缺少"可抄作业"的硬货
- 两者都低 → 换标题重发
